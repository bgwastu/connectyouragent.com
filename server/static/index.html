Connect Your Agent
Source Code

Gives any AI agent temporary shell access to a machine. Create a session, run one command, then paste the prompt into Claude Code, Codex, OpenClaw, or any other agent.

Nothing is logged or stored - sessions run in-memory only.

Use Cases
  • Ask AI to check why your server is running out of disk space at 3am.
  • Ask AI to set up a new dev machine while you keep working.
  • Give AI temporary access to a throwaway machine, lab box, or recovery environment.
  • Debug your home router without pasting screenshots into chats.
Supported: macOS, Windows x64, Linux, OpenWrt / MIPS routers.
